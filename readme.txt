

Neovim to my liking:

Neovim:
    Linux:
        sudo apt-get install neovim
    Windows (pre-build):
        https://github.com/neovim/neovim/releases 

Clone this repo into location:
    Windows: #HOME/AppData/Local/nvim/
    Linux: $HOME/.config/nvim/init.vim

Vim-plug
    Run script from github.com/junegunn/vim-plug

C++ Code completion
    1. get nodejs
        apt-get install nodejs
    2. get Coc 
        :PlugInstall 
    3. get homebrew 
        (see https://docs.brew.sh/Homebrew-on-Linux)
    4. get ccls 
        brew install ccls
